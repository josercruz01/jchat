# jchat
Chat simulator

#### Cloning and running
First check the Dependencies below.

Running file:

```bash
git clone https://github.com/josercruz01/jchat
cd jchat
./run.sh <input_file>
```

Running tests:

```bash
git clone https://github.com/josercruz01/jchat
cd jchat
./test.sh
```



## Dependencies

#### Storm ORM:
ORM for python.

```bash
pip install peewee
```
